# AJE MARTINEZ: Multidisciplinary Artist Portfolio

This project is a high-performance, design-led portfolio and workspace for **Aje Martínez**, a multidisciplinary artist specializing in **Photography**, **Music**, and creative experimentation.

Built with **Next.js 15 (App Router)**, **TypeScript**, and **TailwindCSS**, this platform is designed to showcase high-fidelity media assets with a premium aesthetic.

## Project Structure Overview

Based on our established standard for artistic projects, the following directory structure is implemented:

- `src/app/`: Core application routing and layout.
- `public/content/photography/`: High-resolution visual works and series.
- `public/content/music/`: Original compositions, audio experiments, and samples.
- `src/content/biography/`: Curated artist profiles, statements, and manifestos.
- `docs/projects/`: Internal project briefs, contracts, and planning documentation.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [TailwindCSS](https://tailwindcss.com) & Modern CSS
- **Media**: Optimized for professional-grade photography and audio playback
- **Deployment**: Optimized for Vercel

## Development Workflow

1.  **Installation**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Media Assets**: Place new photography in `public/content/photography` and audio files in `public/content/music`.
